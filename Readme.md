# KitchenSink

![Platform](https://img.shields.io/badge/Platform-iOS%20%7C%20macOS-blue)
![Swift](https://img.shields.io/badge/Language-Swift-orange)
![License](https://img.shields.io/badge/License-MIT-green)

**KitchenSink** ist eine Multiplattform-App (iOS & macOS), die als interaktive Bibliothek für UI-Elemente dient. Sie ermöglicht Entwicklern und Designern, native UI-Komponenten zu durchsuchen und deren Verhalten sowie Darstellung in Echtzeit zu überprüfen.

<p align="center">
  <img src="KitchenSink/Assets.xcassets/AppIcon.appiconset/Appicon.png" alt="KitchenSink App Icon" width="150">
</p>

## Features

* **Komponenten-Browser:** Eine umfassende Übersicht aller gängigen UI-Elemente (Buttons, Toggles, Slider, Textfelder, Listen, etc.).
* **Multiplattform:** Optimiert für iPad, iPhone und Mac.
* **Live-Preview:** Sofortiges Feedback bei Interaktion mit den Elementen.
* **Native Look & Feel:** Nutzt Standard-Systemkomponenten für authentische Darstellung.
* **Light & Dark Mode Support:** Überprüfung der Elemente in beiden Farbschemata.

## Voraussetzungen

Um das Projekt zu bauen und auszuführen, benötigst du:

* **Xcode 26.0** oder neuer
* **iOS 26.0+** / **macOS 26.0+**

## Installation

1.  Klone das Repository:
    ```bash
    git clone [https://github.com/DEIN_USERNAME/KitchenSink.git](https://github.com/inevs/KitchenSink.git)
    ```
2.  Öffne das Projekt in Xcode:
    ```bash
    cd KitchenSink
    open KitchenSink.xcodeproj
    ```
3.  Wähle dein Zielgerät (Simulator oder echtes Gerät) und drücke `Cmd + R` zum Starten.

## Mitwirken

Beiträge sind willkommen! Wenn du ein UI-Element vermisst oder einen Bug findest:

1.  Forke das Repository.
2.  Erstelle einen neuen Branch (`git checkout -b feature/NeuesElement`).
3.  Committe deine Änderungen (`git commit -m 'Add new Toggle Style'`).
4.  Pushe auf den Branch (`git push origin feature/NeuesElement`).
5.  Öffne einen Pull Request.
